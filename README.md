# Overwatch Map Winrate Tracker

Full-stack project scaffold for tracking and analyzing Overwatch map win rates.

## Purpose

This repository is the initial clean structure for a future application that will:

- Collect match outcomes by map
- Calculate win rate trends over time
- Provide a frontend dashboard for map-level performance insights

No application features are implemented yet.

## Planned Tech Stack

- Frontend: React + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
- Caching: Redis
- Containerization: Docker (planned, not set up yet)
- Cloud: AWS (planned)
- Auth: JWT authentication (planned, not implemented yet)
- Testing: Jest
